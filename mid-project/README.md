# 🧑‍💻 React Users Management App

This is a React mid-project built as part of a college assignment, designed to showcase CRUD operations, routing, conditional rendering, and data manipulation using public APIs. This project is suitable for portfolio and resume display.

## 🔧 Features

- Fetch users from JSONPlaceholder API
- Display user data with task completion status (green/red border)
- Search functionality by name or email
- Mouseover reveals additional user data
- Select user to highlight and show todos + posts
- Add, edit, and delete users
- Mark todos as completed
- Add new todo or post for each user

## 🚀 Technologies Used

- React
- React Router
- Fetch API
- JSONPlaceholder (https://jsonplaceholder.typicode.com)
- CSS Modules / custom styles

## 📦 How to Run

1. Clone the repository  
2. Run `npm install` to install dependencies  
3. Run `npm run dev` to start the dev server
